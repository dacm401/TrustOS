# TrustOS Private Beta — User Guide

**Welcome to TrustOS Private Beta!** 🧪

TrustOS 是一个 AI 助手，可以帮你写代码、生成网页、回答问题。你正在参与内测阶段，你的反馈对我们非常重要。

---

## 快速开始

### 基本使用
1. 在聊天框输入你的问题或任务
2. 按 Enter 发送
3. 等待 AI 回复（简单问题几秒，复杂任务可能需要 1-2 分钟）

### 你可以做什么
- **写代码**: "写一个 Python 函数检查回文字符串"
- **生成网页**: "帮我做一个登录页面，包含邮箱、密码和登录按钮"
- **回答问题**: "数据库索引有哪些类型？"
- **改写文案**: "把这段话改得更专业一些"

### 你不能做什么（已知限制）
- ❌ 获取实时数据（天气、股价、新闻等）
- ❌ 访问外部网站或 API
- ❌ 执行代码（只能生成代码，不能运行）

---

## 如何提供反馈

每次 AI 回复后，你会在消息下方看到两个按钮：

- **👍 有帮助** — 回答准确、有用
- **👎 没帮助** — 回答不准确或不符合预期

点击 👎 后，会弹出输入框，**建议填写具体原因**（例如"回答不准确""格式不对""太慢了"），这能帮我们快速定位问题。

**反馈是可选的**，但不反馈我们就不知道哪里需要改进 😊

---

## 常见问题

### Q: 回复太慢了怎么办？
A: 复杂任务（如生成完整网页）可能需要 1-2 分钟。如果超过 2 分钟还没回复，请刷新页面重试，并在反馈中说明。

### Q: 生成的代码/网页不对怎么办？
A: 点击 👎 并说明期望的结果，我们会根据反馈改进模型。

### Q: 可以连续对话吗？
A: 目前每个问题是独立处理的，暂不支持多轮上下文对话。

### Q: 我的数据安全吗？
A: 我们只收集你的反馈（👍/👎 + 可选原因），不会存储你的个人身份信息。反馈原因中请不要填写个人敏感信息。
